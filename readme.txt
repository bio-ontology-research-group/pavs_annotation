step 1. Use the following sparql query to get the pavs data:

step 2. save the data in pavs_final.tsv

step 3. run "sh generate.pavs.custom.data.sh 38" to generate pavs custom data in vcf format to be with VEP for variant annotations. The input parameter is the assembly number. Replace "38" with 37, if you like to generate pavs custom data for assembly Grch37.
